Zadanie 3: # rownanie x'(t) = ax - x^3
############
Punkty równowagi to pierwiastki równania ax-x^3=0 równe: x=0, x=sqrt(a) oraz x=-sqrt(a). 

Dla rownania otrzymujemy macierz Jacobiego postaci : a-3x^2.

Dla pierwszego przypadku x = 0, wyrazenie przyjmuje postac f(x) = a, dla a < 0 punkt równowagi asymptotycznie stabilny.

Dla przypadku drugiego x = sqrt(a), wyrazenie przyjmuje postac f(x) = -2a, dla a>0 jest asymptotycznie stabilny.

Analogicznie dla trzeciego przypadku x = -sqrt(a), wyrazenie przyjmuje postac f(x) = -2a, i tutaj powtarza sie sytuacja, przy wartosciach a>0 mamy asymptotycznie stabilne punkty rownowagi.
